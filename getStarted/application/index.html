课程：详细剖析 "Hello World!" 应用

您已经见过 "Hello World!" 应用 (甚至编译和运行过它)，您可能很好奇它是如何工作的。这是它的代码：


class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

"Hello World!" 应用包含三部分：源码注释， HelloWorldApp 类定义，和 main 方法。 下面的说明帮助您初步理解代码，当您读完剩余的教程后会对代码有更深入的理解。

源码注释

下面的粗体字定义了 "Hello World!" 应用的 注释：

/**
 * The HelloWorldApp class implements an application that
 * simply prints "Hello World!" to standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

注释对程序员来说很有用，但是编译器会忽略它。Java 编程语言支持三种类型的注释：

/* text */
编译器忽略从 /* 到 */ 的所有内容。
/** documentation */
这是文档注释。编译器会忽略这种注释，就像忽略使用 /* 和 */ 的注释。javadoc 工具使用文档注释生成文档。更多信息，查看 Javadoc™ 工具文档。
// text
编译器忽略从 // 到行尾的所有内容。

HelloWorldApp 类定义

下面的粗体文本是 "Hello World!" 应用的类定义：

/**
 * The HelloWorldApp class implements an application that
 * simply displays "Hello World!" to the standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

如上所示，最基本的类定义形如：

class name {
    . . .
}

关键字 class 开启了 name 类的定义，每个类的代码出现在标记为粗体的开闭大括号之间。第 2 章提供了对类的概述，第 4 章详细讨论类。 现在，知道每个应用都是由类定义开始的就够了。

main 方法

下面的粗体文本定义了 main 方法：

/**
 * The HelloWorldApp class implements an application that
 * simply displays "Hello World!" to the standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); //Display the string.
    }
}

在 Java 语言中，每个应用都必须包含如下签名的 main 方法：

public static void main(String[] args)

public 和 static 可以按任意顺序书写 (public static 或者 static public)， 但通常使用 public static。您也可以给参数起任意的名称，但大多数程序员会使用 "args" 或 "argv"。

main 方法很像 C 和 C++ 中的 main 函数，它是应用程序的入口点，接着调用程序所需的其它方法。

main 方法接收一个参数：String 类型的数组组。

public static void main(String[] args)

运行时系统使用数组向您的应用传递信息。比如：

java MyApp arg1 arg2

数组中的每个字符串叫做 命令行参数。命令行参数让用户无需重新编译应用便可影响应用的操作。比如，一个排序程序可能允许用户指定数据降序排列，使用如下的命令行参数：

-descending

"Hello World!" 应用忽略命令行参数，但是您应该意识到有这样的参数存在。

最后，下面这行：

System.out.println("Hello World!");

使用核心库的 System 类向标准输出打印 "Hello World!"。库 (叫做 "Application Programming Interface"，或 "API") 的部分内容将在后续教程中讨论。
