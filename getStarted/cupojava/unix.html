针对 Solaris OS，Linux 和 Mac OS X 的 "Hello World!"

是时候编写您的首个应用了！这些操作是针对使用 Solaris OS，Linux 和 Mac OS X 的用户的。其它平台的指令见 针对 Microsoft Windows 的 "Hello World!" 和 针对 NetBeans IDE 的 "Hello World!"。

如果您在使用本页操作时遇到了问题，参阅 常见问题 (和解决方案)。

核对表 

要编写您的首个程序，需要：

  1. Java SE 8 开发工具 (JDK 8)

    您可以 下载 Solaris OS，Linux 或 Mac OS X 版的 JDK。 (确保您下载的是 JDK，而不是 JRE。) 参见 安装指南。

  2. 文本编辑器

    在本例中，我们将使用 Pico，这是一款很多基于 UNIX 的平台自带的编辑器。如果您使用其它的文本编辑器，比如 vi 或 emacs， 也可以轻松地采用这些操作指令。

这就是您编写首个应用所需的全部。

创建您的首个应用

您的首个应用 HelloWorldApp，仅简单地显示问候语 "Hello World!"。为创建这个程序，您需要：

  • 创建源文件

    源文件包含用 Java 语言写的代码。您可以使用任何文本编辑器创建和编辑源文件。

  • 将源文件编译为 .class 文件

    Java 编程语言 编译器 (javac) 接收源文件并将其转为 JVM 可以理解的指令。.class 文件中包含的指令叫做 字节码。

  • 运行程序

    Java 应用 启动工具 (java) 使用 JVM 运行您的应用。

创建源文件

要创建源文件，您有两个选项：

  • 您可以将文件 HelloWorldApp.java 保存到您的电脑上。然后，直接 编译源文件。

  • 或者，您可以使用下面的操作指令。

首先，打开 shell 或 "终端" 窗口。

新打开的终端窗口。

当前目录 通常是您的 主目录。您可以随时将当前目录切换为主目录，通过键入 cd 然后点击 Return。

您创建的源文件应保存在一个单独的目录中。您可以使用 mkdir 命令创建目录。比如，在 /tmp 目录中创建 examples/java 目录，使用下面的命令：

cd /tmp
mkdir examples
cd examples
mkdir java

要将当前目录改为这个新建的目录，您需要输入：

cd /tmp/examples/java

现在，您可以开始创建源文件了。

通过键入 pico 并点击 Return 启动 Pico 编辑器。如果系统提示 pico: command not found，大概率是 Pico 不可用。 您需要请教系统管理员，或者使用另一种编辑器。

当您启动了 Pico，它将显示一个新的、空白的 buffer。在这里您可以键入代码。

键入下面的代码：

/**
 * The HelloWorldApp class implements an application that
 * simply prints "Hello World!" to standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

输入时要注意   

注意： 输入的所有代码，注释和文件名，包括编译器 (javac) 和启动器 (java) 都是大小写敏感的。

HelloWorldApp 是 不同于 helloworldapp 的。

将代码保存在名称为 HelloWorldApp.java 的文件中。在 Pico 编辑器中，输入 Ctrl-O，然后在底部会提示 File Name to write:，输入文件将要在的保存目录，后跟 HelloWorldApp.java。比如，如果您要将文件 HelloWorldApp.java 保存在 /tmp/examples/java 目录中，就输入 /tmp/examples/java/HelloWorldApp.java 并按 Return。

您可以输入 Ctrl-X 退出 Pico。

将源文件编译为 .class 文件

打开另一个 shell 窗口。将当前目录改为源文件所在的目录。比如，如果您的源文件在 /tmp/examples/java 目录总，输入以下命令并按 Return：

cd /tmp/examples/java

输入 pwd，您将看到当前目录。在本例中以及变为 /tmp/examples/java。

输入 ls 您将看到：

ls 命令结果，显示 .java 源文件。

现在，已经准备好编译源文件了，输入以下命令并按 Return。

javac HelloWorldApp.java

编译器生成了字节码文件 HelloWorldApp.class。输入 ls 将看到新生成的文件： 如下图。

ls 命令结果，显示生成的 .class 文件。

现在，有了 .class 文件，就可以运行程序了。

如果您在这一步遇到了问题， 查阅 常见问题 (和解决方案).

运行程序

在相同的目录，输入：

java HelloWorldApp

下图是您应该看到的结果。

打印 "Hello World!" 到屏幕。

恭喜！您的程序可以正常工作！

如果您在这一步遇到了问题， 查阅 常见问题 (和解决方案).
