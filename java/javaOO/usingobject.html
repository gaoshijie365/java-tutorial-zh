使用对象

一旦创建了对象，您可能想用它来做点什么。您可能想使用其中一个字段的值，改变其中一个字段的值，或者调用其中一个方法来执行一个操作。

引用对象的字段

对象字段是通过它们的名称访问的。您必须使用无歧义的名称。

您可能在自己的类中使用一个简单名。比如，我们可以在 Rectangle 类中添加一条语句，打印 width 和 height：

System.out.println("Width and height are: " + width + ", " + height);

这种情况下，width 和 height 是简单名。

对象的类外部的代码必须使用对象引用或表达式，后跟点 (.) 操作符，然后跟一个简单字段名，如：

objectReference.fieldName

比如，CreateObjectDemo 类中的代码是在 Rectangle 类之外的。所以在 Rectangle 对象 rectOne 中使用 来引用 origin、width 和 height 字段。而在 CreateObjectDemo 类中必须使用 rectOne.origin、 rectOne.width 和 rectone.height 来引用。这个程序使用其中的两个名字来显示 rectOne 的 width 和 height ：

System.out.println("Width of rectOne: "  + rectOne.width);
System.out.println("Height of rectOne: " + rectOne.height);

尝试在 CreateObjectDemo 的代码中使用简单名 width 和 height 是没有意义的 — 这些字段仅在对象中存在 — 并会导致编译错误。

然后，这个程序使用相似的代码展示 rectTwo 的信息。相同类型的字段有它们自己的相同实例字段的拷贝。因此，每一个 Rectangle 对象都有字段名 origin、 width 和 height。当通过对象引用访问实例字段时，您引用的是特定对象的字段。CreateObjectDemo 程序中的两个对象 rectOne 和 rectTwo 拥有不同的 origin、width 和 height 字段。

为访问一个字段，您可以使用指向对象的引用名，就像前例一样，或者也可以使用一个返回对象引用的表达式。new 操作符返回一个对象的引用。所以您可以使用 new 返回的值访问对象的字段：

int height = new Rectangle().height;

这条语句创建了一个新的 Rectangle 对象并立即获取其高度。本质上，这条语句获取的是 Rectangle 的默认高度。 这条语句执行之后，程序不再持有对创建的 Rectangle 对象的引用，因为程序没有存储这个引用。 对象没有被引用，其资源就会被 JVM 回收。

调用对象的方法

使用对象引用调用对象的方法。对象引用后跟点操作符 (.)，然后跟方法的简单名。同时在小括号内提供方法的实参。如果方法不要求任何参数，就使用空的小括号。

objectReference.methodName(argumentList);

或者：

objectReference.methodName();

Rectangle 类有两个方法：getArea() 计算矩形的面积，move() 方法改变矩形的原点。这里的 CreateObjectDemo 代码调用这两个方法：

System.out.println("Area of rectOne: " + rectOne.getArea());
...
rectTwo.move(40, 72);

第一条语句调用 rectOne 的 getArea() 方法并显示结果。第二行移动 rectTwo，因为 move() 方法给对象的 origin.x 和 origin.y 赋了新值。

和实例字段一样，objectReference 必须是对象的引用。您可以使用变量名，也可以使用返回对象引用的任何表达式。 new 操作符返回一个对象引用，所以您可以使用 new 返回的值调用新对象的方法：

new Rectangle(100, 50).getArea()

表达式 newRectangle(100, 50) 返回一个指向 Rectangle 对象的引用。如上所示，您可以使用点符号调用 Rectangle 的 getArea() 方法来计算新矩形的面积。

有些方法，比如 getArea()，返回一个值。对于返回一个值的方法，您可以在表达式中使用方法调用。您可以将返回值赋值给一个变量，使用它做决策，或控制循环。 这段代码将 getArea() 返回的值赋值给变量 areaOfRectangle：

int areaOfRectangle = new Rectangle(100, 50).getArea();

记住，调用特定对象的方法与向该对象发送信息相同。这种情况下，调用的是构造器返回的矩形的 getArea() 方法。

垃圾收集器

一些面向对象的语言要求您跟踪所有您创建的对象，并在不再需要时显式地销毁它们。显式管理内存是乏味且易出错的。Java 平台允许您创建任意数量的对象 (当然，受限于您系统的处理能力)， 并且不用担心销毁它们。当 Java 运行时环境检测到对象不再使用时会删除它们。这个过程叫做 垃圾收集。

当没有任何引用指向对象时，这个对象会被垃圾收集选中。变量持有的引用通常会在变量超出作用域时丢弃。或者，您也可以通过设置变量为 null 显式地丢弃对象引用。 记住，可以有多个引用指向同一个对象，所有指向对象的引用必须全部被丢弃才会被垃圾收集选中。

Java 运行时有一个垃圾收集器，周期性地释放对象占用的内存。垃圾收集器会在会在合适的时间自动地执行回收任务。
