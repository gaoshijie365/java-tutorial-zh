数组

数组 是一个容器，包含同类型的固定数量的值。数组在创建时其长度就已确定。创建之后，其长度就固定下来了。您已经见过数组使用的例子了， 在 "Hello World!" 应用的 main 方法中。本节将更加详细地讨论数组。

10 个元素的数组。


数组中的每一项叫做 元素，每个元素通过其数字 索引 访问。就像上图所示，索引从 0 开始。第 9 个元素，索引为 8。

下面的程序， ArrayDemo， 创建了一个整型数组，往数组中放入一些值，并将每个值打印到标准输出。


class ArrayDemo {
    public static void main(String[] args) {
        // declares an array of integers
        int[] anArray;

        // allocates memory for 10 integers
        anArray = new int[10];
           
        // initialize first element
        anArray[0] = 100;
        // initialize second element
        anArray[1] = 200;
        // and so forth
        anArray[2] = 300;
        anArray[3] = 400;
        anArray[4] = 500;
        anArray[5] = 600;
        anArray[6] = 700;
        anArray[7] = 800;
        anArray[8] = 900;
        anArray[9] = 1000;

        System.out.println("Element at index 0: "
                           + anArray[0]);
        System.out.println("Element at index 1: "
                           + anArray[1]);
        System.out.println("Element at index 2: "
                           + anArray[2]);
        System.out.println("Element at index 3: "
                           + anArray[3]);
        System.out.println("Element at index 4: "
                           + anArray[4]);
        System.out.println("Element at index 5: "
                           + anArray[5]);
        System.out.println("Element at index 6: "
                           + anArray[6]);
        System.out.println("Element at index 7: "
                           + anArray[7]);
        System.out.println("Element at index 8: "
                           + anArray[8]);
        System.out.println("Element at index 9: "
                           + anArray[9]);
    }
}

程序的输出是：

Element at index 0: 100
Element at index 1: 200
Element at index 2: 300
Element at index 3: 400
Element at index 4: 500
Element at index 5: 600
Element at index 6: 700
Element at index 7: 800
Element at index 8: 900
Element at index 9: 1000

在真实的编程场景下，您大概会使用某种 循环结构 来迭代数组中的每个元素。然而，这个例子清晰地说明了数组的语法。您将在 控制流 一节学习几种循环结构 (for、while 和 do-while)。

声明指向数组的变量

前一个程序使用下面一行代码声明了一个数组 (叫做 anArray)：

// declares an array of integers
int[] anArray;

和声明其它类型的变量一样，数组的声明也包含两部分：数组类型和数组名称。数组的类型写做 type[]，其中 type 是其包含的元素的类型。中括号表明这个变量持有数组。数组的大小不是类型的一部分 (这也就是中括号是空的的原因)。 数组的名称可以是任意的，只要遵循之前的命名规则和习惯就行了。和其它类型的变量一样，声明并没有真的创建数组，它仅仅是告诉编译器这个变量持有特定类型的数组。

类似地，您也可以声明其它类型的数组：

byte[] anArrayOfBytes;
short[] anArrayOfShorts;
long[] anArrayOfLongs;
float[] anArrayOfFloats;
double[] anArrayOfDoubles;
boolean[] anArrayOfBooleans;
char[] anArrayOfChars;
String[] anArrayOfStrings;

您也可以将中括号放在数组名称后：

// this form is discouraged
float anArrayOfFloats[];

然而，并不推荐这种写法，中括号表明是数组类型，应该和类型出现在一起。

创建、初始化和访问数组

一种创建数组的方式是使用 new 操作符。ArrayDemo 中的下一条语句分配了一个数组，有足够的内存容纳 10 个整型元素，并将其赋值给变量 anArray。

// create an array of integers
anArray = new int[10];

如果没有这条语句，编译器将报告如下的错误：

ArrayDemo.java:4: Variable anArray may not have been initialized.

下面几行给数组的每个元素赋值：

anArray[0] = 100; // initialize first element
anArray[1] = 200; // initialize second element
anArray[2] = 300; // and so forth

通过数字索引访问每一个元素：

System.out.println("Element 1 at index 0: " + anArray[0]);
System.out.println("Element 2 at index 1: " + anArray[1]);
System.out.println("Element 3 at index 2: " + anArray[2]);

可选地，您也可以使用更简短的语法来创建和初始化数组：

int[] anArray = { 
    100, 200, 300,
    400, 500, 600, 
    700, 800, 900, 1000
};

数组的长度由大括号中值的数量决定。

您也可以声明数组的数组 (也叫做 多维 数组)，通过使用多对中括号，比如 String[][] names。必须通过对应的索引访问每个元素。

在 Java 中，多维数组是一个数组，其组件本身是数组。这和 C 或 Fortran 中的数组不同。这么做的后果是每一行的长度可以不同，如下面程序 MultiDimArrayDemo 所示：

class MultiDimArrayDemo {
    public static void main(String[] args) {
        String[][] names = {
            {"Mr. ", "Mrs. ", "Ms. "},
            {"Smith", "Jones"}
        };
        // Mr. Smith
        System.out.println(names[0][0] + names[1][0]);
        // Ms. Jones
        System.out.println(names[0][2] + names[1][1]);
    }
}

程序的输出是：

Mr. Smith
Ms. Jones

最后，您可以使用内置的 length 属性获取数组的大小。下面的代码将数组大小打印到标准输出：

 System.out.println(anArray.length);

复制数组

System 类中有一个 arraycopy 方法，可以高效地将一个数组的数据复制到另一个数组中：

public static void arraycopy(Object src, int srcPos,
                             Object dest, int destPos, int length)

两个 Object 参数分别代表拷贝的源和目的地。三个 int 参数分别代表源数组的起始位置和目标数组的起始位置和要拷贝的元素数量。

下面的程序， ArrayCopyDemo， 声明了一个元素类型为 String 的数组。它使用 System.arraycopy 方法复制了数组的一个子序列到第二个数组中：

 
class ArrayCopyDemo {
    public static void main(String[] args) {
        String[] copyFrom = {
            "Affogato", "Americano", "Cappuccino", "Corretto", "Cortado",   
            "Doppio", "Espresso", "Frappucino", "Freddo", "Lungo", "Macchiato",      
            "Marocchino", "Ristretto" };
        
        String[] copyTo = new String[7];
        System.arraycopy(copyFrom, 2, copyTo, 0, 7);
        for (String coffee : copyTo) {
            System.out.print(coffee + " ");           
        }
    }
}

这个程序的输出是：

Cappuccino Corretto Cortado Doppio Espresso Frappucino Freddo

数组操作

数组是一项强大和有用的概念。Java SE 提供了方法来执行和数组相关的常见操作。比如， ArrayCopyDemo 使用 System 类的 arraycopy 方法，代替手工迭代源数组的每个元素并将其放入目标数组。这是在幕后执行的，使开发者仅用一行代码就可完成复制操作。

为方便开发，Java SE 在 java.util.Arrays 类中提供了一些方法执行数组操作 (常见的任务，比如复制、排序和搜索)。 比如，上个例子可以使用 java.util.Arrays 类的 copyOfRange 进行改写，如 ArrayCopyOfDemo 所示。不同之处在于 copyOfRange 方法并不要求您在调用方法之前创建目标数组，因为目标数组由方法返回：


class ArrayCopyOfDemo {
    public static void main(String[] args) {
        String[] copyFrom = {
            "Affogato", "Americano", "Cappuccino", "Corretto", "Cortado",   
            "Doppio", "Espresso", "Frappucino", "Freddo", "Lungo", "Macchiato",      
            "Marocchino", "Ristretto" };
        
        String[] copyTo = java.util.Arrays.copyOfRange(copyFrom, 2, 9);        
        for (String coffee : copyTo) {
            System.out.print(coffee + " ");           
        }            
    }
}

正如您所看到的，程序的输出是相同的，但代码行数更少。copyOfRange 的第二个参数是要拷贝的第一个索引 (包含)，第三个参数是要拷贝的最后一个索引 (不包含)。 在本例中，拷贝的范围不包括索引为 9 的元素 (包含字符串 Lungo)。

java.uti.Arrays 类中提供的其它实用操作有：

  • 给一个值，搜索其在数组中的索引 (binarySearch 方法)。

  • 比较数组是否相等 (equals 方法)。

  • 用同一个值填充数组 (fill 方法)。

  • 升序排序数组。可以使用 sort 方法顺序执行，也可以使用 parallelSort 方法 (Java SE 8 新增) 并行执行。 在多处理器系统上对大数组进行排序比顺序排序快。

  • 使用数组作为源创建流 (stream 方法)。比如，下面语句打印 copyTo 数组的内容：

    java.util.Arrays.stream(copyTo).map(coffee -> coffee + " ").forEach(System.out::print);

    查看 聚合操作 获取关于流的更多信息。

  • 转换数组为字符串。toString 方法将数组中的每个元素转换为字符串，并由逗号分隔它们，然后使用中括号包括它们。 比如，下面的语句将 copyTo 数组转换为数组并打印：

    System.out.println(java.util.Arrays.toString(copyTo));

    这条语句打印如下：

    [Cappuccino, Corretto, Cortado, Doppio, Espresso, Frappucino, Freddo]
