赋值、算术和一元操作符

简单赋值操作符

您将遇到的一个最常见的操作符就是简单赋值操作符 "="。您在 Bicycle 类中见过这个操作符，它将右边的值赋给左边的操作数：

 int cadence = 0;
 int speed = 0;
 int gear = 1;

这个操作符也可以用来给对象赋值 对象引用，正如 创建对象 中所讨论的。

算术操作符

Java 提供了操作符来执行加、减、乘、除运算。它们对应基础的数学运算。只有一个 "%" 符号对您来说是全新的，它做除法运算并返回余数作为结果。

操作符 描述
+加操作符 (也用作字符串拼接)
-减操作符
*乘操作符
/除操作符
%取余操作符

下面的程序， ArithmeticDemo，测试算术操作符。


class ArithmeticDemo {

    public static void main (String[] args) {

        int result = 1 + 2;
        // result is now 3
        System.out.println("1 + 2 = " + result);
        int original_result = result;

        result = result - 1;
        // result is now 2
        System.out.println(original_result + " - 1 = " + result);
        original_result = result;

        result = result * 2;
        // result is now 4
        System.out.println(original_result + " * 2 = " + result);
        original_result = result;

        result = result / 2;
        // result is now 2
        System.out.println(original_result + " / 2 = " + result);
        original_result = result;

        result = result + 8;
        // result is now 10
        System.out.println(original_result + " + 8 = " + result);
        original_result = result;

        result = result % 7;
        // result is now 3
        System.out.println(original_result + " % 7 = " + result);
    }
}

程序打印如下：

1 + 2 = 3
3 - 1 = 2
2 * 2 = 4
4 / 2 = 2
2 + 8 = 10
10 % 7 = 3

您可以刻组合算术操作符和简单赋值操作符，创建 复合赋值。比如 x+=1; 和 x=x+1; 都将 x 的值增加 1。

+ 操作符也可以用来拼接字符串。如下面的 ConcatDemo 所示：


class ConcatDemo {
    public static void main(String[] args){
        String firstString = "This is";
        String secondString = " a concatenated string.";
        String thirdString = firstString+secondString;
        System.out.println(thirdString);
    }
}

程序的最后，变量 thirdString 包含 "This is a concatenated string."，并打印到标准输出。

一元操作符

一元操作符只能有一个操作数。它们可以执行一些操作，如增加/减少 1，对表达式取反，反转布尔值。

操作符 描述
+一元加操作符。表示正数值 (然而，数字不加这个也表示正数)
-一元减操作符。对表达式取反
++自增操作符。将值增加 1
--自减操作符。将值减少 1
!逻辑取反操作符。将布尔值取反

下面的程序， UnaryDemo 测试一元操作符：


class UnaryDemo {

    public static void main(String[] args) {

        int result = +1;
        // result is now 1
        System.out.println(result);

        result--;
        // result is now 0
        System.out.println(result);

        result++;
        // result is now 1
        System.out.println(result);

        result = -result;
        // result is now -1
        System.out.println(result);

        boolean success = false;
        // false
        System.out.println(success);
        // true
        System.out.println(!success);
    }
}

自增/自减操作符可以放在操作数的前面(前缀)或后面(后缀)。result++; 和 ++result; 都会使 result 的值加 1。 不同之处是，前缀版 (++result) 取增加后的值，后缀版 (result++) 取原始值。如果您只执行简单的自增/自减，选择哪个版本都可以。 但是如果您将其用作更大表达式的一部分，两者将会有明显的不同。

下面的程序， PrePostDemo， 演示了前缀/后缀一元自增操作：


class PrePostDemo {
    public static void main(String[] args){
        int i = 3;
        i++;
        // prints 4
        System.out.println(i);
        ++i;			   
        // prints 5
        System.out.println(i);
        // prints 6
        System.out.println(++i);
        // prints 6
        System.out.println(i++);
        // prints 7
        System.out.println(i);
    }
}
